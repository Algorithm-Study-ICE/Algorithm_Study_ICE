## ***Rule***
- 매주 목요일 자정까지 `팀원이 1문제`씩 선별 후 각자 풀어보기.
- 매주 회의는 수요일 `21시`
- `백준`, `프로그래머스`, `LeetCode` 문제에서 선별하기.
- 주마다 레파지토리에 문제 풀기
- 무단으로 회의 불참시 `랜덤 카카오 선물`하기.
